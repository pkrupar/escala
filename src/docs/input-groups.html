    Input group with buttons

    Buttons in input groups are really easy, just add <button class="button"></button> or <a class="button"></a> inside .inputGroup. Buttons with class .button are by default outlined (no need to add additional .button--outlined), this is because filled buttons would attract too much attention in contrast with subtle text fields.

    I recommend to only use .button--secondary, however .button--primary will totally work, but it's discouraged due to it's high visual importance.
