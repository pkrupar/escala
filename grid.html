Grid

    Standart 12 column responsive grid (very similar to Bootstrap grid). The biggest difference is that there're no columns .col and rows .row instead there're grid cells .grid-cell and grids .grid. Any row could be multiline, for that reason grid is more appropriate name.

    Responsivness

    • grid-cell--xs-{1-12}
    • grid-cell--sm-{1-12}
    • grid-cell--md-{1-12}
    • grid-cell--lg-{1-12}
    • grid-cell--xl-{1-12}

    Important: when you use e.g: grid-cell--md-6, grid-cell--xs-12 should be also used to create full width layout on tiny viewports because fluid width is default behaviour for grid cells.
